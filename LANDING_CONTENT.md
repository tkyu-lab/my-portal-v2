## 2. 学習ロードマップセクション

### セクションタイトル
**「プロへの3ステップ：AIが土台を作り、あなたが命を吹き込む。」**

### 説明文
Web制作のプロフェッショナルになるための学習パスは、3つの明確なステップで構成されています。各ステップは前のステップの上に構築され、最終的には自立した開発者としての基礎を完成させます。

### ステップ1: AI Foundation
**「Manusを利用し、自然言語プロンプトからフルスタックのサイトを数分で生成する方法を習得。」**

このステップでは、AIの力を最大限に活用して、プロトタイプから本番レベルのサイトまで、迅速に構築するスキルを身につけます。Manusのプロンプト設計、テンプレート選択、初期カスタマイズなど、AIアシスタントを効果的に指揮するテクニックを学びます。

**学習内容：**
- Manusプロンプト設計のベストプラクティス
- テンプレート選択とカスタマイズ基礎
- AIが生成したコードの初期評価と改善点の特定
- プロジェクト構成とファイル管理

### ステップ2: The Refinement Layer
**「エクスポートしたコードをGitHubへ同期し、セマンティックHTMLへの修正や、CSS Grid/Flexboxによるレイアウトの安定化を学ぶ。」**

AIが生成したコードは優れていますが、本当のプロフェッショナルは手作業で磨き上げます。このステップでは、コードの品質を段階的に向上させるテクニックを習得します。セマンティックHTML、モダンCSSレイアウト、アクセシビリティ対応など、実務レベルの改善スキルを身につけます。

**学習内容：**
- セマンティックHTMLの実装
- CSS Grid/Flexboxによるレイアウト最適化
- レスポンシブデザインの実装
- アクセシビリティ（WCAG）への対応
- パフォーマンス最適化（Lighthouse対応）
- GitHub連携と手動編集履歴の管理

### ステップ3: Interactive Logic
**「JavaScriptによるDOM操作や、Stripe、Google Mapsなどの外部API連携による『テンプレート感』の払拭を学ぶ。」**

最後のステップは、インタラクティブな機能の実装です。JavaScriptを駆使して、ユーザー体験を豊かにし、外部サービスとの連携を実現します。このステップを完了すれば、あなたは完全に自立したWeb制作者として、クライアント案件に対応できるレベルに到達します。

**学習内容：**
- JavaScriptの基礎から応用
- DOM操作とイベントハンドリング
- 非同期処理（Promise、async/await）
- 外部API連携（Stripe決済、Google Maps、Weather API等）
- 状態管理とコンポーネント設計
- デバッグとエラーハンドリング

## 3. 学習ログセクション（Build in Public）

### セクションタイトル
**「挑戦の軌跡が、信頼という名のポートフォリオになる。」**

### 説明文
学習の過程を公開することは、単なる記録ではありません。それは、あなたの成長を証明し、クライアントや採用企業に対する信頼を構築する最強のポートフォリオになります。AI-to-Manualでは、スクリーンショットとテキストで進捗を投稿でき、コミュニティ全体で学習成果を共有できます。

### 機能説明
- **画像投稿**: 完成したサイトのスクリーンショット、コード改善前後の比較画像をアップロード
- **テキスト説明**: 学習内容、工夫した点、困った点などを記述
- **言語タグ**: HTML、CSS、JavaScript、React、API連携など、使用した技術をタグ付け
- **コミュニティ反応**: 他のユーザーからの「いいね」やコメントで、モチベーションを維持
- **ポートフォリオ化**: 投稿履歴は自動的にあなたの学習ポートフォリオとなり、プロフィールページで公開可能

### タイムライン表示
各投稿はシックなカードデザインで表示され、時系列で整理されます。カードには投稿者の名前、投稿日時、使用技術タグ、いいね数が表示され、クリックで詳細を確認できます。

## 4. AI適応型トレーニング

### セクションタイトル
**「あなたの限界にAIが挑む。習熟度に応じたカスタム・チャレンジ。」**

### 説明文
単なる問題集ではなく、AIがあなたの習熟度をリアルタイムで判定し、最適な難易度の問題を出題するシステムです。初級から上級まで、段階的にスキルを磨くことで、確実な成長を実感できます。

### 難易度レベル

**初級（Weight 1）**
- 変数や基本的なHTMLタグの穴埋め問題
- CSSの基本プロパティ（color、font-size、margin等）の理解
- JavaScriptの基本構文（変数宣言、条件分岐、ループ）
- 所要時間: 5～10分

**中級（Weight 2）**
- ループ処理やFlexboxの配置修正
- 複雑なセレクタやCSSの応用
- JavaScriptの関数設計とスコープ理解
- 簡単なAPI呼び出し
- 所要時間: 15～30分

**上級（Weight 3）**
- 非同期処理やAPI連携のデバッグ
- 複雑なJavaScriptロジックの実装
- パフォーマンス最適化の実践
- セキュリティを考慮した実装
- 所要時間: 30～60分

### ダッシュボード機能
- **習熟度円グラフ**: 初級・中級・上級の完了率を視覚化
- **進捗追跡**: 完了した問題数、連続学習日数、マスタリースコア
- **推奨問題**: AIが次に解くべき問題を提案
- **解答履歴**: 過去の解答と改善点を確認

## 5. 質問コーナー（ソクラテス式メンター）

### セクションタイトル
**「答えではなく、導きを。AIがヒントを与え、仲間が解決を助ける。」**

### 説明文
わからないことがあったとき、単に答えを教えてもらうのではなく、考え方を導いてもらうことが、真の学習につながります。AI-to-Manualの質問フォーラムは、AIが最初にソクラテス式のヒントを提示し、その後、コミュニティメンバーが議論を通じて解決を支援する仕組みになっています。

### フォーラム機能

**質問投稿**
- タイトル、詳細説明、コード例（オプション）を投稿
- カテゴリ選択（HTML、CSS、JavaScript、API、デプロイ等）
- 画像やコードスニペットの添付

**AIによるソクラテス式ヒント**
- 質問投稿直後、AIが「考え方のヒント」を自動生成
- 直接的な答えではなく、問題解決のプロセスを示唆
- ユーザーが自分で答えにたどり着くための導き

**コミュニティ回答**
- 他のユーザーが経験や知識を共有
- 複数の解決方法が提案される可能性
- ベストアンサー機能で、最も役立つ回答を表示

**解決マーク**
- 問題が解決したら、質問者が「解決済み」とマーク
- 解決済みスレッドはシルバーのチェックマークで表示
- 他のユーザーが参考にしやすい

### カテゴリ例
- HTML/セマンティック
- CSS/レイアウト
- JavaScript/DOM操作
- API連携
- デプロイ/運用
- パフォーマンス
- セキュリティ

## 6. 運用・保守・更新モジュール

### セクションタイトル
**「納品はスタートに過ぎない。継続的な信頼を築く運用スキルを磨く。」**

### 説明文
フリーランスとして継続案件を獲得するには、サイト公開後の運用・保守スキルが不可欠です。このモジュールは、日々のタスク管理から月次の監査まで、プロフェッショナルな運用体制を構築するための知識と実践を提供します。

### 運用タスク管理

**デイリータスク**
- セキュリティログの確認
- バックアップ状態の確認
- エラーログの監視

**ウィークリータスク**
- パフォーマンス監視（Lighthouse実行）
- SEOチェック（メタタグ、構造化データ）
- ユーザーフィードバック確認

**マンスリータスク**
- セキュリティパッチの適用
- 依存ライブラリの更新確認
- アナリティクスレビュー
- バックアップの整合性確認

### カレンダー表示
- 色分けされたタスク表示（完了：緑、セキュリティ：青、SEO：オレンジ、保留中：赤）
- ドラッグ&ドロップでタスク日程を調整
- 完了チェックで進捗管理

### SEO最適化ガイド
- Lighthouse監査の実施と改善方法
- Schema.orgの構造化データ実装
- メタタグの最適化
- Core Web Vitals対応

### セキュリティ管理
- SSL証明書の更新管理
- セキュリティヘッダー（CSP、X-Frame-Options等）の設定
- 依存ライブラリの脆弱性チェック
- バックアップ戦略

| Responsive Champion | レスポンシブ問題を15問完了 | モバイルファースト設計の習得 |
| Git Guru | GitHub同期で10回以上の手動編集 | バージョン管理とコラボレーション |
| Performance Optimizer | パフォーマンス最適化問題を10問完了 | 高速化とLighthouse対応 |

## 8. GitHub同期ダッシュボード

### セクションタイトル
**「AIと手作業の共同作業を可視化する。」**

### 説明文
Manusで生成したコードをGitHubにエクスポートし、その後の手動修正を追跡することで、AIと人間の協働プロセスを記録できます。このダッシュボードは、あなたの改善スキルを可視化し、ポートフォリオとしての価値を高めます。

### ダッシュボード機能

**リポジトリ管理**
- 接続されたGitHubリポジトリの一覧表示
- リポジトリごとのAI生成日時と最終更新日時
- コミット数とプルリクエスト数の統計

**手動編集履歴**
- AI生成後の手動編集数を可視化
- 編集内容の分類（HTML改善、CSS最適化、JavaScript追加等）
- 改善前後のコード比較表示

**改善トレンド**
- 時系列で改善活動を追跡
- 改善の質と量を評価
- 習熟度の向上を数値化

**エクスポート機能**
- 現在のプロジェクトをGitHubにワンクリックエクスポート
- 既存リポジトリへのプッシュ
- コミットメッセージの自動生成

## 9. クライアント対応シミュレーター

### セクションタイトル
**「フリーランスに必須のスキルを、安全に練習する。」**

### 説明文
Web制作スキルと同じくらい重要なのが、クライアント対応スキルです。契約書作成、見積もり提示、プロジェクト管理など、ビジネス面でのスキルを実践的に学べるシミュレーターです。

### シミュレーター機能

**プロジェクト作成**
- クライアント名、プロジェクトタイトル、説明を入力
- プロジェクト規模（小規模、中規模、大規模）を選択
- 要件定義書のテンプレートを提供

**見積もり作成**
- 工数見積もり（時給制、固定料金制を選択）
- 項目別の見積もり内訳
- 割引やオプション料金の追加
- 自動計算で総額を表示

**契約書生成**
- テンプレートから契約書を自動生成
- 支払い条件、納期、保守期間などを設定
- PDF出力機能

**プロジェクト管理**
- マイルストーン設定
- 進捗状況の記録
- クライアント対応メモ

**シミュレーション結果**
- 見積もり精度の評価
- 実際のプロジェクト相場との比較
- 改善提案の提示
